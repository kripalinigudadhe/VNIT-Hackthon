🍱 FoodBank – VNIT Hackathon Project
📌 Overview
FoodBank is a web-based platform developed during the VNIT Hackathon to reduce food wastage and help distribute excess food to people in need.
The platform connects donors (restaurants, individuals) with recipients (NGOs, shelters, needy individuals) in a simple and efficient way.

🚀 Features
👤 User Authentication
Login & Signup functionality

🍲 Food Donation
Donors can add details of surplus food

📥 Request System
Recipients can request available food

📊 Dashboard
Overview of donations and requests

🤝 Matching System
Connects donors with recipients

🧾 Donation Tracking
Keeps record of food donations

This project aims to:
